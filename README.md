# arduino-morse(Arduino学习)

### 第一次作业 morse
#### 提交时间: 7月3日晚上
从Arduino串口读取一个字符串，并将其转换为莫尔斯码，以便从数据接口输出。使用前请将Morse文件夹复制到Arduino的libraries目录中。
#### 文件结构
+ morse.ino 主程序
+ Morse
    + keywords.txt 关键字说明
    + Morse.cpp Morse电码输出类
    + Morse.h Morse电码输出类的头文件
